# Web Development – Level 1 Projects

This repository contains three beginner‑level web development projects to practice HTML, CSS, and JavaScript.

## 📁 Projects
1. **Personal Portfolio**
   - Build a personal portfolio website using HTML & CSS.
   - Sections: Header (name/logo & tagline), About (image + bio), Skills, Projects (with titles, descriptions, images), Resume (PDF link), Contact (email & phone), and Footer (copyright).
   - Focus on clean layout, styling, and responsive design.

2. **Landing Page** Task 2
   - Create an attractive landing page with HTML & CSS.
   - Learn to structure columns, sections, headers & footers.
   - Emphasize visual design: alignments, padding, color palette, and CSS box model to avoid overlapping elements.

3. **Basic Calculator** Task 3
   - Develop a simple calculator using HTML, CSS, and JavaScript.
   - Features: display screen, buttons for +, −, ×, ÷ operations.
   - Use CSS grid for layout and JavaScript event listeners, if‑else, and operators to handle calculations.
